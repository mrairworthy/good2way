good2way

Offline FIR/UIR phrasebook and aviation reference app.

good2way is an offline-focused FIR/UIR phrasebook for pilots. It helps you quickly find FIRs and UIRs, save favorites, organize them into groups, add personal notes, and use aviation tools such as coordinate conversion.

good2way is built to support familiarisation, briefing and day-to-day aviation preparation — while elevating communication skills and embracing kindness between aviators.

Features

  • Offline-focused FIR/UIR phrasebook
  • Search by FIR name, identifier, country, area, language, ACC or note content
  • Favorites and custom groups
  • Private notes per FIR with UTC-based timestamps
  • Coordinate converter for common WGS84 formats, UTM and MGRS
  • Built-in manual, QRH, glossary and changelog
  • Optional regional database unlocks via Apple In-App Purchase

Important disclaimer

good2way provides non-binding, reference-only information on FIR/UIR greetings, phrase examples and related metadata. It is intended for familiarisation and convenience only.

For operational use, always consult current official sources, including ICAO provisions, State publications such as AIP, AIC and NOTAM, and applicable operator documentation.

good2way is not a flight planning, navigation, performance, fuel, weather, terrain, obstacle or ATC-clearance planning tool.
